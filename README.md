# Harald

A macOS app that displays bluetooth peripherals. Built using a Combine layer on top of CoreBluetooth.

## Requirements

* macOS 10.15
* Xcode 11.3
* Swift 5

## Installation

```
git clone git@github.com:andyshep/Harald.git
open Harald/Harald.xcodeproj
```
